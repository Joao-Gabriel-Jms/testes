Galeria de Gatos

Projeto de uma galeria interativa de gatos desenvolvido para praticar habilidades de Front-end.

Sobre o projeto

Este projeto consiste em um site que exibe diferentes categorias de gatos, permitindo ao usuário navegar e visualizar imagens de forma organizada e interativa.

Foi desenvolvido com foco em aprendizado e aplicação de conceitos fundamentais de desenvolvimento web.

Funcionalidades
Exibição de imagens de gatos
Sistema de filtragem por categorias
Interface simples e intuitiva
Layout responsivo (adaptado para celular)
Tecnologias utilizadas
HTML5
CSS3
JavaScript
Git e GitHub
Figma (para prototipação)
Responsividade

O site foi desenvolvido utilizando conceitos de mobile-first, garantindo boa visualização em dispositivos móveis.

Objetivo

Praticar:

Estruturação de páginas com HTML
Estilização com CSS
Interatividade com JavaScript
Organização de projetos
Versionamento com Git
